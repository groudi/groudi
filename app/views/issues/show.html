Guidelines

What to do in this page?

  • The matrix table given above is a discision table(your decision) for the discussion undergoing.
  • The first row is the set of available options which are going to be evaluated on the basis of some specific criterion(1st column).
  • Each criterion has a predefined weight.
  • You evaluate the cell and provide a reasonable vote from 1-5.
  • The vote is multiplied with the weight to calcualte the points for that cell. After all the grids are filled, you can see the resulting sum at the bottom.
  • You can comment on each cell by double clicking on the cell input.
